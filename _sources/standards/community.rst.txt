Community
=========

The LabCAS community for NIST consists of the users, developers, administrators, and other stakeholders.

- Please view our `Code of Conduct`_.
- If you'd like to help, please check out our `Contributing Guidelines`_.

Software developed for the NIST community is released as open source under the `Apache 2.0 License`_. Some software may contain secrets and is not released as open source.


Repositories
------------

The following software organizations on GitHub_ comprise the LabCAS system for NIST:

- `Labcas-NIST <https://github.com/Labcas-NIST>`_ is the main organization of LabCAS software for NIST.
- `JPL LabCAS <https://github.com/jpl-labcas>`_ contains the base LabCAS software developed by JPL.
- `EDRN <https://github.com/EDRN>`_ contains the LabCAS software developed by the Early Detection Research Network, which still contains some LabCAS origins.
