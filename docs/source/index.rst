JPL NIST LabCAS
===============

Welcome to the documentation portal for the `Laboratory and Catalog Archive System`_ for the `National Institutes of Standards and Technology`_.

LabCAS is a system for managing and sharing laboratory data and metadata. It is a web-based application that allows users to create, manage, and share laboratory data and metadata. LabCAS for NIST is a custom implementation of the LabCAS system for the National Institutes of Standards and Technology, enabling NIST to manage and share its scientific data for cell line provenance, flow cytometry, genome editing, micriobial data, and more.
